# Índice de Agentes — Equipe de Conteúdo

Tabela de roteamento. Vic consulta isto antes de delegar qualquer pedido.

| Especialista | Pasta | Aciona quando o pedido é... |
|---|---|---|
| Vic | `Equipe/Vic - Editor-Chefe/AGENTS.md` | Aprovação final, decisão editorial, conflito entre especialistas, "publique isso" |
| Kai | `Equipe/Kai - Repórter de Notícias/AGENTS.md` | "o que saiu de novo hoje", "tem notícia sobre X", varredura diária de lançamentos/patches/eventos |
| Vera | `Equipe/Vera - Curadora de Notas/AGENTS.md` | "qual a nota de X", "atualiza os scores", agregação de crítica externa |
| Theo | `Equipe/Theo - Redator de Reviews/AGENTS.md` | "escreve a review de X", redação de texto longo seguindo a rubrica de `types.ts` |
| Dara | `Equipe/Dara - Arquiteta de Dados/AGENTS.md` | Qualquer coisa que vai tocar em `src/lib/data.ts`, `types.ts`, ou qualquer URL de imagem/vídeo |

Bootstrap: se este índice encolher abaixo de 3 especialistas, reative o processo de contratação (adaptar [[../../../Downloads/WeWiki-Alunos-main/Wiki equipe/SOPs/SOP-001-como-adicionar-novo-especialista]] se precisar de um 6º especialista, ex. um Moderador de Comentários).
